# Supertrend 优化记录

## 当前研究基线

2026-07-18 起，策略重新收敛为可解释的 Supertrend 基线：

```text
3m Supertrend 翻转或回踩收回
  -> 大周期背景过滤
  -> 趋势、均线、MACD、量价和风险过滤
  -> 全仓开仓
```

Supertrend 是唯一信号源，其他信息只能过滤或记录。历史 SMC、shock 和 intraday 独立入口实验已经删除。

## 已确认的历史问题

- 允许 SMC BOS 独立触发时，交易序列高度集中于单一辅助信号，策略不再具有 Supertrend 可解释性。
- 放宽 shock 或日内脉冲入口会显著增加交易和手续费，但没有形成稳定正期望。
- 多个方向指标累加评分会重复表达相同趋势状态，旧评分没有观察到稳定的收益排序能力。
- 入口和退出必须继续采用单变量实验，不能同时修改。

旧回测结果、ClickHouse Run 和临时报告已经清理。它们只用于促成当前语义修正，不再作为现版本收益依据。

## 下一步实验

1. 运行 `supertrend_flip` 与 `wick_reclaim` 共同启用的年度基线。
2. 按触发来源、方向、大周期背景和退出原因拆分交易。
3. 如果交易量不足，先分别回测翻转与回踩，不引入新的独立指标入口。
4. 固定入口后再比较 `structure`、`trailing` 和 `adaptive` 出场。
5. 最后才评估 1m 执行择时；1m 不得参与策略判断。

所有版本在完成年度、重点行情和样本外验证前均不能进入实盘。

## 市场特征与事件基础设施计划

后续优化优先建设可复用的市场能力层，不继续在策略内部堆叠指标判断。数据链路统一为：

```text
K线与成交数据
  -> 连续特征 Feature
  -> 市场状态 State
  -> 市场事件 Event
  -> 质量评估 Quality
  -> 策略授权与执行 Strategy
```

基础设施只描述市场，不产生买卖决策。Supertrend 继续作为当前策略唯一的方向授权事件；其他特征、状态和事件只能参与质量评估、过滤、风险控制或离线研究。

### 能力域

| 能力 | 连续特征 | 状态与事件 |
| --- | --- | --- |
| 趋势 | 方向、ADX、DI 差、斜率、持续时间、趋势效率 | 形成、延续、成熟、衰减、冲突 |
| 波动 | ATR 百分比、ATR 分位数、波幅变化率、带宽 | 收缩、正常、扩张、异常、冲击 |
| 结构 | 摆动高低点、HH/HL、LH/LL、区间边界 | BOS、CHoCH、结构延续、结构失效 |
| 位置 | 均线 ATR 距离、布林位置、VWAP 偏离、摆动点距离 | 回撤区、价值区、区间边缘、过度延伸 |
| 事件 | 穿越幅度、突破幅度、回踩距离、影线收回 | 翻转、突破、回踩、reclaim、假突破、均线交叉 |
| 质量 | 年龄、确认数、连续性、冲突数、来源族群 | 新鲜度、可靠度、拥挤度、冲突度 |

### 分层语义

连续值、离散状态和瞬时事件必须分开。例如 `adx14=28.6` 是 Feature，`trend_strength=strengthening` 是 State，`trend_strength_confirmed` 才是 Event。策略消费稳定语义，不直接重复解释一组原始指标。

每项基础能力至少应声明：

- 稳定字段名、类型、单位和计算周期；
- 当前值、前值、变化量、斜率或分位数；
- 数据有效性、口径版本和计算时间；
- 来源族群、方向、首次发生时间、年龄和失效状态；
- 是否严格只使用当时可见数据。

### 实施阶段

1. **能力审计与数据字典**：盘点现有指标输出、消费者、单位、窗口、缺失条件、重复语义和未来数据风险。
2. **统一特征协议**：在现有 `ValueSet`、数值窗口和信号窗口上增量扩展，不先重构指标模块。
3. **市场状态层**：基于历史分布生成趋势、波动、结构和位置状态，避免凭单次回测硬编码阈值。
4. **事件生命周期**：统一记录方向、幅度、发生时间、年龄、确认数、失效状态和来源。
5. **质量与冲突层**：按趋势、波动、位置、结构和成交量族群限制重复贡献，避免相关指标重复计分。
6. **研究与标签层**：统一生成 30/60/120 分钟方向收益、MFE、MAE、最大回吐和 first-passage 标签。
7. **策略接入**：只有通过按月及样本外稳定性验证的能力才能进入评分；仍不得绕过 Supertrend 独立授权交易。

### 当前数据结论

2025-09-01 至 2025-12-01 的 ETHUSDT 3m 候选信号研究表明，多数指标存在非线性甜区，不能采用“越强越好”的评分方式：

- 中等 ATR 百分比的方向收益和路径质量优于极端波动；
- Supertrend 距离、STC 变化和 VWAP 偏离过大时，信号更接近过度扩张或趋势末端；
- ADX 和 EMA 斜率极高不能直接视为高质量入口；
- 评分应拆分为 `direction_score`、`entry_quality_score` 和 `overextension_penalty`。

当前 `entry_feature_snapshot` 已随 Supertrend 候选事件记录已有趋势、波动、位置和动量特征，用于离线研究；该诊断不参与策略评分或过滤。

### 验证边界

- 实时、回测和研究必须使用同一特征口径；
- 不因单次策略盈亏临时修改基础字段含义；
- 先验证底层数据的区分能力，再评估策略收益；
- 训练窗口内完成按月稳定性检验后，必须更换时间段做样本外交叉验证；
- 1m 仅在 3m 信号出现后用于执行择时，不参与策略方向判断。
