# Supertrend 优化记录

## 当前研究基线

2026-07-18 起，策略重新收敛为可解释的 Supertrend 基线：

```text
3m Supertrend 翻转或回踩收回
  -> 大周期背景过滤
  -> 趋势、均线、MACD、量价和风险过滤
  -> 全仓开仓
```

Supertrend 是唯一信号源，其他信息只能过滤或记录。历史 SMC、shock 和 intraday 独立入口实验已经删除。

## 2026-07-19 研究收敛结论

当前可复现的单持仓研究基线固定为：初始保证金 `10000 USDT`、单笔保证金 `100 USDT`、`100x` 杠杆、初始止损 `50 bps`、盈利 `50 bps` 后抬至 `+16 bps`、盈利 `75 bps` 后按 `30 bps` 回撤全仓退出，往返手续费与滑点合计 `16 bps`。研究比较标准、自适应和 AI 三种 Supertrend 的原始 `flip`，不把实验性延续事件称为正式策略信号。

ETHUSDT 2025-08 的基线结果为：标准版 `38` 笔、`-424.09 USDT`、PF `0.643`；自适应版 `38` 笔、`-413.78 USDT`、PF `0.652`；AI 版 `15` 笔、`+19.68 USDT`、PF `1.050`。AI 只是小样本微利，尚不能证明跨月正期望；标准和自适应在该高波动月份明显亏损。

逐笔诊断确认，基线亏损主要是未达到保本门槛前触发初始止损，不是大幅盈利后的回吐。标准版有多笔盈利退出后继续运行，说明退出存在提前兑现问题，但把全仓跟踪参数放宽到 `100/40` 或 `150/60` 后只改善8月，2025-09至11月跨月结果恶化，属于行情依赖，不保留。

以下增加交易次数的实验已判定失败并从代码删除：同趋势结构重入、独立3m BOS立即/次根确认、5m BOS后3m回踩收复。它们虽能增加成交，但新增交易为负期望。AI候选因子上限从 `1-5` 缩到 `1-4` 或 `1-3.5` 能让首次翻转更早、覆盖更多波段，却显著增加震荡翻转；两档PF分别降至 `0.259` 和 `0.454`，同样删除。

因此当前不再把“提高单一标的单月订单数”作为优化目标。后续只有在全部 AI flip 的统一前向标签中找到预先定义、跨窗口稳定的被拦截正期望子集，才允许增加入口；否则保持原始 AI `1-5` 的选择性。

## 已确认的历史问题

- 允许 SMC BOS 独立触发时，交易序列高度集中于单一辅助信号，策略不再具有 Supertrend 可解释性。
- 放宽 shock 或日内脉冲入口会显著增加交易和手续费，但没有形成稳定正期望。
- 多个方向指标累加评分会重复表达相同趋势状态，旧评分没有观察到稳定的收益排序能力。
- 入口和退出必须继续采用单变量实验，不能同时修改。

旧回测结果、ClickHouse Run 和临时报告已经清理。它们只用于促成当前语义修正，不再作为现版本收益依据。

## 下一步实验

1. 对全部 AI `1-5` flip 统一计算未来 `20/40/80` 根3m K线的 MFE、MAE、收盘收益和 first-passage 标签，包括已放行与被市场状态拦截的信号。
2. 只检验预先声明的候选维度，例如5m/15m方向、同向BOS/CHoCH、量能和动量扩张；不能根据单月盈利交易反推阈值。
3. 候选必须先提高被拦截信号的净期望，再验证加入单持仓回放后的PF、回撤和成本；增加订单数本身不是验收条件。
4. 固定入口后再研究退出，不同时修改入口与退出；全仓退出参数必须通过逐月和冻结窗口验证。
5. 最后才评估1m执行择时；1m不得参与策略方向判断。

所有版本在完成年度、重点行情和样本外验证前均不能进入实盘。

## 市场特征与事件基础设施计划

后续优化优先建设可复用的市场能力层，不继续在策略内部堆叠指标判断。数据链路统一为：

```text
K线与成交数据
  -> 连续特征 Feature
  -> 市场状态 State
  -> 市场事件 Event
  -> 质量评估 Quality
  -> 策略授权与执行 Strategy
```

基础设施只描述市场，不产生买卖决策。Supertrend 继续作为当前策略唯一的方向授权事件；其他特征、状态和事件只能参与质量评估、过滤、风险控制或离线研究。

### 能力域

| 能力 | 连续特征 | 状态与事件 |
| --- | --- | --- |
| 趋势 | 方向、ADX、DI 差、斜率、持续时间、趋势效率 | 形成、延续、成熟、衰减、冲突 |
| 波动 | ATR 百分比、ATR 分位数、波幅变化率、带宽 | 收缩、正常、扩张、异常、冲击 |
| 结构 | 摆动高低点、HH/HL、LH/LL、区间边界 | BOS、CHoCH、结构延续、结构失效 |
| 位置 | 均线 ATR 距离、布林位置、VWAP 偏离、摆动点距离 | 回撤区、价值区、区间边缘、过度延伸 |
| 事件 | 穿越幅度、突破幅度、回踩距离、影线收回 | 翻转、突破、回踩、reclaim、假突破、均线交叉 |
| 质量 | 年龄、确认数、连续性、冲突数、来源族群 | 新鲜度、可靠度、拥挤度、冲突度 |

### 分层语义

连续值、离散状态和瞬时事件必须分开。例如 `adx14=28.6` 是 Feature，`trend_strength=strengthening` 是 State，`trend_strength_confirmed` 才是 Event。策略消费稳定语义，不直接重复解释一组原始指标。

每项基础能力至少应声明：

- 稳定字段名、类型、单位和计算周期；
- 当前值、前值、变化量、斜率或分位数；
- 数据有效性、口径版本和计算时间；
- 来源族群、方向、首次发生时间、年龄和失效状态；
- 是否严格只使用当时可见数据。

### 实施阶段

1. **能力审计与数据字典**：盘点现有指标输出、消费者、单位、窗口、缺失条件、重复语义和未来数据风险。
2. **统一特征协议**：在现有 `ValueSet`、数值窗口和信号窗口上增量扩展，不先重构指标模块。
3. **市场状态层**：基于历史分布生成趋势、波动、结构和位置状态，避免凭单次回测硬编码阈值。
4. **事件生命周期**：统一记录方向、幅度、发生时间、年龄、确认数、失效状态和来源。
5. **质量与冲突层**：按趋势、波动、位置、结构和成交量族群限制重复贡献，避免相关指标重复计分。
6. **研究与标签层**：统一生成 30/60/120 分钟方向收益、MFE、MAE、最大回吐和 first-passage 标签。
7. **策略接入**：只有通过按月及样本外稳定性验证的能力才能进入评分；仍不得绕过 Supertrend 独立授权交易。

### 当前数据结论

2025-09-01 至 2025-12-01 的 ETHUSDT 3m 候选信号研究表明，多数指标存在非线性甜区，不能采用“越强越好”的评分方式：

- 中等 ATR 百分比的方向收益和路径质量优于极端波动；
- Supertrend 距离、STC 变化和 VWAP 偏离过大时，信号更接近过度扩张或趋势末端；
- ADX 和 EMA 斜率极高不能直接视为高质量入口；
- 评分应拆分为 `direction_score`、`entry_quality_score` 和 `overextension_penalty`。

当前 `entry_feature_snapshot` 已随 Supertrend 候选事件记录已有趋势、波动、位置和动量特征，用于离线研究；该诊断不参与策略评分或过滤。

### 验证边界

- 实时、回测和研究必须使用同一特征口径；
- 不因单次策略盈亏临时修改基础字段含义；
- 先验证底层数据的区分能力，再评估策略收益；
- 训练窗口内完成按月稳定性检验后，必须更换时间段做样本外交叉验证；
- 1m 仅在 3m 信号出现后用于执行择时，不参与策略方向判断。

### 统计口径

- 原始样本为 3,273 个 Supertrend 翻转候选；每个候选独立模拟，允许持仓时间重叠。
- 当前策略最终放行 588 个信号。原始候选与当前放行信号分别统计，不能混称为实际订单。

### 指标有效性分析

指标使用方向归一化值：买入方向保持原值，卖出方向对有方向的距离和动量取反。例如 `dynamic_swing_vwap_distance_pct <= -0.137658` 表示买入时价格至少低于动态 Swing VWAP 约 0.138%，卖出时价格至少高于动态 Swing VWAP 约 0.138%，即价格处在信号方向的回撤侧。

单指标按 20% 分位阈值扫描，并在六种退出轮廓上分别比较三个窗口的净期望。当前结果分为三类：

| 结论 | 指标或状态 | 结果 |
| --- | --- | --- |
| 稳定改善但不足以单独盈利 | `market_strength_state == strong` | 六种退出中五种在三个窗口同时改善；500% / 70% 平均提升 6.9 个百分点，但过滤后仍为 -2.8% |
| 有效辅助条件 | `adx14 <= 16.2934` | 中低趋势强度对中高止盈更有利；单独使用仍为负期望 |
| 有效辅助条件 | `atr_pct14 <= 0.203708` | 主要改善 500% 大波段捕获；单独使用的 500% / 70% 为 -4.2% |
| 有效辅助条件 | `dynamic_swing_vwap_distance_pct <= -0.137658` | 回撤侧入场改善多个退出轮廓，但单独使用的 500% / 70% 为 -4.5% |
| 弱改善 | `market_location_score <= 0`、较高市场强度与动量 | 多数窗口改善，但不能独立产生正期望 |
| 暂无稳定增量 | STC 区域、EMA alignment、原始方向 bias | 单字段差异小或跨窗口反向，不能据此设硬过滤 |

最值得继续验证的是低波动与回撤位置的交互，而不是某个单指标：

```text
atr_pct14 <= 0.203708
AND directional_dynamic_swing_vwap_distance_pct <= -0.137658
```

该组合保留 364 个原始信号，三个窗口样本数分别为 98、139、127；当前策略只放行其中 50 个，即只捕获 13.7%。这支持“有利润空间的信号没有被现有评分充分发掘”的判断。

| 500% / 70% | 364 | 18 | 227 | 119 | 32.4% | +4.29% | 1.078 | 1.008 | +10.40% / +0.45% / +3.79% |
| 500% / 150% | 364 | 21 | 125 | 218 | 44.5% | +7.96% | 1.116 | 1.006 | +17.84% / +0.40% / +8.60% |

这是本轮唯一在三个窗口均达到正净期望和 Profit Factor 大于 1 的指标组合，但不能直接升级为策略规则：阈值来自同一批研究窗口，最弱窗口 PF 仅约 1.01，优势非常薄，且 500% 目标依赖少数大波段。它当前只能标记为 `Promising / requires OOS validation`。

ADX 与回撤位置也有明显交互。`adx14 <= 16.2934` 且处于动态 Swing VWAP 回撤侧时保留 202 个信号，在六种退出轮廓中全部跨窗口改善；100% / 100% 和 500% / 70% 的合并平均已转正，但仍有单个窗口为负，因此稳定性弱于上述低 ATR 组合。

当前数据不支持“指标越强越好”。相反，更有效的结构是：Supertrend 刚翻转时市场总体能力不弱，但短周期 ADX、ATR 不宜过高，入场价格应位于动态 Swing VWAP 的回撤侧。它描述的是低波动回撤后的潜在延续，而不是高波动追涨杀跌。

下一步必须固定上述两个候选组合，不再扫描阈值，并使用未参与本轮研究的新时间窗口做样本外验证；通过后才能考虑修改入场过滤。

## ETH 2025-08 至 2025-11 信号研究协议

下一阶段不执行策略仓位，只研究 ETHUSDT 3m 在 `2025-08-01T00:00:00Z` 至 `2025-12-01T00:00:00Z` 之间的全部 `supertrend_flip` 和 `wick_reclaim` 原始触发。每个信号相互独立，从信号后的下一根 K 线开始按时间回放，最长观察 12 小时。

每个信号同时计算固定保证金止损 `50% / 70% / 100% / 150%`、ATR 止损 `1.0 / 1.5 / 2.0 / 2.5 ATR`，以及保证金止盈阶梯 `30% / 50% / 75% / 100% / 150% / 200% / 300% / 500%`。杠杆口径固定为 100 倍，同时保存换算后的标的价格 bps。若同一根 K 线同时触及止盈和止损，保守地按止损处理；首次止损后停止观察，后续反弹不再属于该信号。

ClickHouse 使用两张研究表：

- `supertrend_signal_research`：每个原始信号一行，保存身份、方向、触发来源、入场价、ATR，以及入场时点的完整底层指标、窗口语义和多周期指标 JSON。
- `supertrend_signal_outcomes`：每个信号、止损模型和止盈阶梯一行，保存首次止盈、首次止损或到期结果，以及退出前 MFE、MAE、最高命中止盈阶梯和观察根数。

研究明细只保存在 ClickHouse；本文档后续只记录止盈止损矩阵、指标分布、稳定性检验和结论，避免把大体量逐信号数据写入 Markdown。训练区间内发现的阈值必须固定后再换区间验证，不能直接升级为策略规则。

## 趋势平台延续事件研究

2026-07-19 增加研究专用的 `trend_platform_breakout` 事件。该事件不进入在线 Supertrend 策略，不授权开仓，也不修改仓位或执行配置。研究检测器要求：

- 3m 与 5m Supertrend 方向一致；
- 收盘价同时位于 3m 与 5m Supertrend 价格线的趋势侧；
- 此前固定数量的 3m K 线形成窄幅平台；
- 当前收盘突破平台高点或低点；
- 当前成交量相对平台平均量达到最低倍数；
- 同方向事件在冷却窗口内只记录一次。

生命周期为 `inactive -> building -> ready -> breakout -> consumed/invalidated`。研究命令通过 `platform-window-bars`、`platform-max-range-pct`、`platform-min-volume-ratio` 和 `platform-cooldown-bars` 参数显式控制口径，事件参数和当时的 3m/5m Supertrend 状态写入 `trigger_metadata_json`。

ETHUSDT 2026-07-13 00:00 至 2026-07-19 07:30 的初次回放使用 12 根平台窗口、0.6% 最大振幅、1.5 倍成交量和 20 根冷却窗口，共记录 62 个平台事件，其中 53 个没有同时出现原始 Supertrend 翻转。该事件捕获了 2026-07-19 00:51 的平台后启动，但在 2 小时观察、固定 50% 保证金止盈、70% 保证金止损及 16% 往返成本口径下，多空平均净收益均为负。

因此当前结论是：平台延续属于有效的新事件观测能力，但现有四个条件不足以形成可交易入口。后续只能先做更长区间、first-passage 和样本外研究，不得直接接入策略授权。

### 最近七天多事件覆盖研究

同日继续增加两个研究事件，仍不进入在线策略：

- `volatility_impulse`：短窗口方向位移达到 ATR 倍数、收盘突破近期结构且 `volume_ratio20` 放大，用于捕获没有预先平台的急涨急跌。
- `trend_pullback_resume`：3m 与 5m Supertrend 同向、价格保持在两条 Supertrend 线趋势侧，回踩 EMA25 后在限定时间内重新突破短期结构且 `volume_ratio5` 恢复，用于捕获趋势二次启动。

2026-07-13 00:00 至 2026-07-19 07:30 的组合回放共保存 224 个研究信号。按截图中八段主要多空波段人工固定启动窗口，`supertrend_flip`、`trend_platform_breakout`、`volatility_impulse` 和 `trend_pullback_resume` 的联合事件在每段启动前后 45 分钟内均找到同方向候选；该结果只表示事件覆盖，不表示可交易收益。

净收益使用 2 小时观察、固定 50% 保证金止盈、70% 保证金止损和 16% 往返成本。三类事件都提高了大波段覆盖，但候选数量和假动作同步增加，当前没有任何一类达到正净期望。因此联合覆盖结果不得解释为“最近七天八段行情都可以盈利交易”，下一阶段应研究事件去重、过度延伸、剩余空间和同类假信号标签，而不是继续增加入口。

### 跨来源同向去重

在同一根 3m K 线上先合并全部事件来源，再对合并后的买卖方向分别应用 20 根完整 K 线冷却。买卖方向的冷却互相独立；冷却只减少重复研究样本，不改变各检测器的事件定义，也不接入在线交易授权。

相同七天区间复跑后，研究信号由 224 个降至 127 个，减少 43.3%。八段人工固定波段仍全部在启动前后 45 分钟内找到同方向候选，最近事件距离人工启动点为 1 至 29 分钟。在相同 2 小时、50% 止盈、70% 止损和 16% 往返成本口径下，去重前后整体平均净收益均为 -21.16%。

因此 20 根冷却可以作为研究数据降噪规则：它在本区间保留 8/8 覆盖且没有降低平均结果，但没有改善负期望。现有入场特征在盈利与亏损候选之间高度重叠，当前证据不足以冻结“过度延伸”或“剩余空间”硬阈值；为避免在七天样本上追拟合，本轮不把这两项写成交易过滤器。

### 2026-05-01 至 2026-06-08 逆势门控消融

针对 ETHUSDT 从 2422.52 下跌至 1503.60、期间包含多轮反弹的行情，增加研究专用逆势门控：15m 或 30m 任一方向与信号一致时按顺势处理；两个周期都明确反向时，要求等待 3 根 3m K 线、5m Supertrend 已翻转、突破最近 3 根结构且 `volume_ratio20 >= 1.5`，每个高周期趋势阶段只允许一次，并在元数据中标记 25% 仓位因子。该逻辑不进入在线策略。

组合事件回放保存 534 个信号，其中顺势 512 个、逆势 22 个。九段人工固定多空波段仍全部有同方向候选，但首个候选延迟扩大至 96 至 643 分钟。使用固定 150% 止盈、150% 止损、30 分钟退出后冷却，并对逆势结果按 25% 因子缩放时，组合事件顺序回放为 104 笔，累计净结果 -4303.26%，最大回撤 4303.26%。

只保留与旧研究同源的 `supertrend_flip` 做消融时，新门控为 79 笔、累计净结果 -886.04%、最大回撤 1953.70%；未加该门控的基线为 106 笔、累计净结果 +1173.75%、最大回撤 826.99%。新门控虽然减少逆势尝试，却同时删除了过多有利的早期反转，并显著增加信号延迟，结论为失败，不得进入实盘。后续如继续研究，应使用独立样本先定义“高周期趋势衰竭”标签，而不是在本窗口继续调整等待根数、量比或结构阈值。

### 入场后 3 至 5 根 K 线有效性观察

同一区间增加研究观察表 `supertrend_signal_validation_observations`，对每个候选保存第 3、4、5 根 3m K 线时的 MFE、MAE、方向收盘收益、5m Supertrend 方向，以及收盘是否曾反穿信号 K 线高低点。观察能力与在线退出完全隔离，失败的逆势门控默认关闭。

本轮组合事件共有 796 个候选和 2388 条检查点观察。以固定 150% 止盈、150% 止损、退出后冷却 30 分钟为基线，枚举 3/4/5 根检查点和 10/15/20 bps 最低 MFE，并在原止盈止损尚未先触发时模拟提前退出。所有九种组合均为负；表现相对较好的 4 根/20 bps 组合仍累计 -2267.87%、最大回撤 2754.73%，同时救下 73 笔原止损并误杀 76 笔原盈利。

单条件消融同样未形成可靠规则：信号 K 线高低点过窄，结构条件对绝大多数顺序入选交易都会触发；5m 反向确认明显滞后；最低 MFE 条件误杀数不低于救回的止损数，例如 3 根/15 bps 救下 95 笔原止损但误杀 114 笔原盈利。结论是这些字段适合作为后验诊断观察，但不能直接用于提前退出。后续若继续，应先定义基于多根 K 线或 ATR 的宽结构失效区，并在独立窗口冻结定义，不能继续围绕本区间扫描阈值。

### 横盘状态识别研究

增加研究专用 `ChopDetector` 和 `supertrend_chop_observations` 观察表，不接入在线策略，也不直接禁止开仓。检测器以最近 60 根 3m K 线为观察窗，对以下五项证据投票：价格效率比、平台宽度/10m ATR、10m ADX、10m Supertrend 翻转次数、10m Supertrend 线的 ATR 归一化斜率。至少三票连续 10 根 3m K 线后进入 `chop_confirmed`；放量突破平台后进入 `breakout_pending`，连续两根保持在平台外才退出；横盘证据明显不足时也必须连续 5 根才退出，避免状态随单根 K 线来回切换。

ETHUSDT 2026-06-28 至 2026-07-15 回放共保存 8,580 条观察。按截图人工固定时段统计，`chop_watch`、`chop_confirmed` 与 `breakout_pending` 合计覆盖：6月29日磨损段 86.2%、7月3日平台 85.0%、7月10日平台 71.1%、7月11日震荡 87.2%、7月14日平台 49.0%；7月14日放量突破段 54 根全部为 `normal`。初版状态没有证据消失退出，7月8日真实下跌段曾有 60 根滞留在 `chop_confirmed`；增加 5 根退出迟滞后降至 35 根，但该趋势段仍有 17.5% K 线被确认成横盘，说明误杀尚未消除。

将同区间 368 个研究信号按状态联表后，直接过滤 `chop_confirmed` 和 `breakout_pending` 未显示正向增量。在固定 50% 止损、75% 止盈口径下，被过滤组 149 个候选的成本前平均结果为 +6.71 bps，允许组 217 个候选为 -4.49 bps；按 16 bps 往返成本换算，两组均为负，但被过滤组反而少亏。固定 150% / 150% 时结论相同：被过滤组成本前 +13.19 bps，允许组 +1.39 bps。横盘标签能描述人工截图中的多数盘整，却不能把“不可交易磨损”与“平台边缘反转/蓄势后延续”可靠分开。

因此当前只保留状态观察和诊断能力，不在策略入口增加硬过滤。下一步应固定定义后换独立区间验证，并增加平台内位置、突破失败次数与信号方向的交互；评价必须使用带手续费、单持仓和退出后冷却的顺序回放，不能用横盘覆盖率替代交易收益。
